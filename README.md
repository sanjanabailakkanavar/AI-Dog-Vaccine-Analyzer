# 🐶 AI Dog Vaccine Analyzer

An AI-powered dog vaccine analysis system that uses OCR and LLM-based intelligence to analyze dog vaccine images and generate intelligent veterinary insights.

The application extracts text from uploaded vaccine images, identifies vaccine names using pattern matching, and enhances the analysis using Generative AI.

---

# 🚀 Features

- ✅ OCR-based vaccine text extraction
- ✅ AI-powered dog vaccine analysis
- ✅ LLM-enhanced veterinary insights
- ✅ Regex-based vaccine detection
- ✅ Streamlit interactive UI
- ✅ Real-time image processing
- ✅ Automated veterinary assistance
- ✅ Intelligent vaccine matching

# 🧠 How It Works

1. User uploads a dog vaccine image
2. EasyOCR extracts text from the image
3. Regex patterns identify vaccine names
4. LLM analyzes extracted information
5. AI generates intelligent veterinary insights
